# LED 闪烁示例

## 功能说明
这个示例演示如何控制 ESP32 开发板上的板载 LED 进行闪烁。

## 硬件要求
- ESP32-WROOM-32 开发板
- Micro-USB 数据线

## 引脚说明
- **LED 引脚**: GPIO2（板载蓝色 LED）

## 代码说明
- 使用 `pinMode()` 设置引脚模式
- 使用 `digitalWrite()` 控制引脚电平
- 使用 `delay()` 进行延时
- 使用串口输出调试信息

## 上传步骤
1. 打开 [blink.ino](blink.ino)
2. 选择正确的开发板: ESP32 Dev Module
3. 选择正确的串口
4. 点击上传按钮
5. 打开串口监视器（波特率 115200）

## 预期结果
- LED 每秒闪烁一次
- 串口监视器输出 LED 状态

## 扩展练习
1. 修改闪烁频率
2. 改变闪烁模式（例如：SOS 信号）
3. 添加多个 LED
