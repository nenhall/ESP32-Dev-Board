# Wi-Fi 连接示例

## 功能说明
这个示例演示如何连接 ESP32 到 Wi-Fi 网络，并显示详细的网络信息。

## 硬件要求
- ESP32-WROOM-32 开发板
- 2.4GHz Wi-Fi 网络（不支持 5GHz）

## 配置步骤
1. 打开 [wifi_connect.ino](wifi_connect.ino)
2. 修改以下配置：
   ```cpp
   const char* ssid = "你的WiFi名称";      // 改为你的 WiFi 名称
   const char* password = "你的WiFi密码";  // 改为你的 WiFi 密码
   ```
3. 保存文件

## 上传步骤
1. 选择开发板: ESP32 Dev Module
2. 选择串口
3. 上传程序
4. 打开串口监视器（波特率 115200）

## 预期结果
连接成功后，串口监视器将显示：
- ✓ Wi-Fi 连接成功!
- SSID（网络名称）
- IP 地址
- 子网掩码
- 网关地址
- DNS 服务器
- 信号强度
- 信道

## 连接失败排查
如果连接失败，程序会显示详细的原因和建议：
1. 检查 WiFi 名称和密码是否正确
2. 确认使用的是 2.4GHz WiFi（不是 5GHz）
3. 检查信号强度
4. 检查路由器的 MAC 过滤设置

## 扩展功能
